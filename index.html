About

My name is Jacob Barrett and I'm a University of Iowa CS student who has worked on projects for Counter-Strike: Global Offensive Kreedz, a mod of the original game. I started with programming back in early high school starting with Bash. Then, a few years passed and I decided to take a Python coding class in college. Since then, I've touched Java, Python, HTML/CSS, Javascript, Sourcepawn, Assembly. However, web dev is what I feel most comfortable with, so I've spent most of my focus on front end development. Below are a few projects that I've made for the Kreedz community. If you have any questions, contact me. Thanks!

Projects

"KREEDZ" is a general index.html file that links all Kreedz projects together. It uses parallax + jquery for-looping. It's currently out of date and I intend on going back and updating it once I finish with other projects.

"Kreedz Guide" is a guide that was originally created by Johnny "JWL" Waffles to teach new players how to KZ. This page takes his videos and includes them along with a transcript. I intend on expanding on it with Advanced KZ Tech + extra notes for the beginner parts.

"Random Map Generator" is as it is aptly named: a random man generator. I made it for Kreedz Twitch streamers to be able to randomly get a map without having to ask chat or without having to scroll through a bunch of pages in-game. I plan on redesigning it later.

"Top Longjumpers", much like Random Map Generator, its named after what it does. It gets the top longjumpers. However, this was created to practice ajax/jquery and API accessing. It also was designed for practice with tables, formatting, and much more. It's still a work in progress and is my current focus.
